# HIM

> A personal Character AI engine built with Python, Ollama, and Qwen3.

## Overview

HIM is a local AI companion inspired by Character.AI.

Unlike a simple chatbot, HIM is designed as a complete character engine with persistent memory, emotions, relationship growth, and personality-driven conversations.

The first character is **Jungkook**, designed as a calm, patient, emotionally intelligent billionaire with a consistent personality.

---

## Features

- Local LLM using Ollama
- Qwen3 integration
- Character configuration system
- Prompt builder
- Modular architecture
- Persistent memory (coming soon)
- Emotion engine (coming soon)
- Relationship engine (coming soon)
- Desktop UI (planned)
- Voice conversations (planned)
